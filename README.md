# AI2000 Docs

Personal knowledge base for AI tips, best practices, Claude Code skills, and MCP servers.

Built with [Zensical](https://zensical.org/).

## Local dev

```bash
uv sync --dev
uv run zensical serve            # localhost:8000
uv run zensical build --clean    # build to site/
```
